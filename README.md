# 614 Site — Dual Brand Homepage

> **614 Restore** (Roofing/Contracting) + **614 Tech** (SaaS Products: TrussCTR, QuoteMGR)

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Vercel-ready deployment

## Getting Started

```bash
npx create-next-app@latest 614-site --typescript --tailwind --app
cd 614-site
npm run dev
```

Replace `app/page.tsx` with `src/app/page.tsx` from this repo.
